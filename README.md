# Prompt Engineering Projects

This repository contains the capstone projects developed as part of the **Prompt Engineering Internship**. Each project demonstrates the application of modern web technologies, backend development, cloud deployment, and AI-powered solutions to solve real-world problems.

## 📁 Projects Included

### 📄 Resume_Job
An AI-powered Resume and Job Matching Platform that helps users analyze resumes and compare them against job descriptions to determine compatibility and generate detailed reports.

### 💬 Support_Assistant
An AI Customer Support Assistant built using React and FastAPI. The application enables users to manage customer conversations, support tickets, and dashboard activities through a modern web interface integrated with Supabase.

## 🛠️ Technologies Used

- React
- Vite
- FastAPI
- Python
- Supabase
- REST APIs
- Vercel
- Render
- Git & GitHub

---

These projects were developed for learning and demonstrating practical skills in full-stack application development, API integration, cloud deployment, and modern software engineering practices.
